# RAWG Game Discovery Platform 

A modern game discovery web app built using the RAWG Video Games Database API. Browse, search, and explore games across platforms, genres, and ratings.

---

## Features

-  Search for games by name
-  Browse games by genre, platform, and popularity
-  View ratings, release dates, and screenshots
-  Detailed game information pages
-  Fast and responsive UI

---

##  Tech Stack

- Frontend: React / Next.js (update if different)
- Styling: CSS / Tailwind / Bootstrap
- API: [RAWG Video Games Database](https://rawg.io/apidocs)
- State Management: (Redux / Context API / etc.)

## Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/rawg-gamehub.git
cd rawg-gamehub
```

2. Install dependencies:
```bash
npm install
```

3. Create a .env file and add your RAWG API key:

```bash
VITE_RAWG_API_KEY=your_api_key_here
```

4. Start the development server:
```bash
npm run dev
